# River data processing addendum — draft

> **Draft only — not legal advice.** A finalized DPA requires counsel, processor agreements, and a real subprocessors list.

The completed DPA should define controller/processor roles, processing instructions, categories of data subjects and personal data, purpose and duration of processing, confidentiality, technical and organizational measures, subprocessors, international transfers, assistance with rights requests and incidents, deletion/return procedures, audit rights, and applicable standard contractual clauses.
